== Description ==

The first true all-in-one GeoCaster solution for WordPress,
specifically designed for businesses who provide services in multiple countries, states and cities.
Place short code [WPGEOCASTER] on you post, page.


== Installation ==

* Upload `wp-geocaster` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Place short code [WPGEOCASTER] on you post, page.
